# My Quantum Computing Journey 
This repository tracks my progress as a Quantum Algorithm/Software Engineer (QASE).

## Project 1: The Bell State (Entanglement)
- **Goal:** To create a quantum link between two qubits.
- **Tools:** Python, Qiskit, Google Colab.
- **Key Logic:** Used a Hadamard gate to create superposition and a CNOT gate for entanglement.
- **Result:** Successfully simulated a 50/50 probability of measuring '00' or '11'.

## Project 2: Quantum Random Number Generator
- **Goal:** To create three qubits which generate a random number from 0 to 7.
- **Tools:** Python, Qiskit, Google Colab.
- **Key Logic:** - Used Hadamard gate to create superposition between three qubits.<br>
                 - Used StatevectorSampler to generate a random binary number.<br>
                 - Converted binary to decimal and displayed the output.<br>
- **Result:** Successfully generated a random number between 0 & 7
